Autonomous Human-Following Mobile Robot

Vision-based person-following robot built on a TurtleBot 4 Lite running ROS 2 Jazzy. Uses YOLOv8 for real-time detection, a custom 4-node pipeline with Kalman-filtered tracking, proportional follow control, and a hardware safety supervisor. Maintains ~1 m following distance with obstacle avoidance on real hardware.

Minidrone Autonomous Landing

Parrot Mambo minidrone that detects a color-marked moving target using its downward camera and autonomously lands on it. A 9-zone binary mask drives a Stateflow state machine for real-time path following and descent. Designed in MATLAB/Simulink with C code generated and deployed directly on hardware.
