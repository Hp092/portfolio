Minidrone Autonomous Landing

Parrot Mambo minidrone that detects a color-marked moving target with its downward camera and autonomously lands on it. A 9-zone binary mask drives a Stateflow state machine for real-time path following and descent. Designed in MATLAB/Simulink with C code deployed on hardware.

MATLAB Simulink Stateflow Computer Vision PID Control Parrot Mambo

End-to-end system that photographs a physical maze, solves it with A*, and commands a 6-DOF Pro600 arm to trace the solution path. Python handles vision and pathfinding; MATLAB runs inverse kinematics and trajectory generation; joint angles are sent over TCP to hardware.
